# 정기 학술대회

키오스 포인트 결제 기준

✅ 0 ~ 10,999점
할인 없음 (기본가 그대로)

✅ 11,000 ~ 21,999점
정기학술대회 = 0원
전야제는 그대로

✅ 22,000 ~ 34,999점
할인 없음 (기본가 그대로)
※ 코드상 이 구간은 “무료” 조건에 포함되지 않음

✅ 35,000 ~ 39,999점
정기학술대회 = 0원
전야제는 그대로

✅ 40,000점 이상
정기학술대회 = 0원
전야제 = 0원
(둘 다 완전 무료)
