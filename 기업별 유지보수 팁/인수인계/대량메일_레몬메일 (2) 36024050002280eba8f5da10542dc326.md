## 주소록 등록

1. 새그룹 만들기
2. csv 파일업로드
3. 예제파일 다운로드
4. 고객사에서 준 명단 이메일 복사 > 예제 엑셀 파일에 붙여넣기
5. 파일업로드
6. 업로드 시작

## 메일 발송 주의사항

- 그룹선택 후 우측에 그룹추가 누르기
- 고객사에서 전달해준 제목 오탈자 확인
- 발송예약 날짜 / 시간 더블체크
- 에디터 내 소스 확인
    - 에디터 파일 등록 시 파일명 영문으로 변경
- 첨부파일 요청 시 첨부파일 정확히 들어갔는지 더블체크
- 수신동의문구 내 날짜 최신화
    - 예약 발송 시 예약한 날짜로 변경
- 미리보기로 에디터 소스(이미지맵 링크, 이미지 등) 잘 들어갔는지 재확인
    - 링크 점검 시 Ctrl 누르고(새창열기) 해야함
        - 현재창 열기 시 세팅한 메일 다 날아감..
