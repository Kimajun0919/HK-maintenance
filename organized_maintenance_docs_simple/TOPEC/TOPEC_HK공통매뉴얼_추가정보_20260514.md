### 5-7. TOPEC 웹진 작업 방법

- 기존에 작업된 사이트 참고하여 고객사에서 전달해 준 이미지 편집
- 토펙 엔지니어링 FTP 정보로 접속: `/www/webzine`
- 아래 경로에 새로운 폴더 생성, 이전에 작업된 파일 복붙해서 넣기

**수정 경로**
1. `/www/webzine/{해당월}/layout/header.php` — 로고 링크 & 해당 월 내용에 맞춰 텍스트 수정
2. `/www/webzine/{해당월}/main/index.php` — 고객사가 전달해 준 내용으로 텍스트, 이미지 수정 (`api` 폴더: 6월호에 사용된 폴더 복사해서 가져오기)
3. `/www/webzine/{해당월}/sub` — 서브 페이지들 수정 (디자인은 크게 신경 쓰지 않아도 OK, 기존 레이아웃 최대한 활용)
4. `/www/webzine/{해당월}/config.php` — 우측 메뉴 / 메인 화면 글 제목 / 지난호 보기 수정 가능

**완성 후**: `http://www.topec.co.kr/` 풋터 위 eTOPIA 버튼에 새로 제작된 웹진 링크 연결
