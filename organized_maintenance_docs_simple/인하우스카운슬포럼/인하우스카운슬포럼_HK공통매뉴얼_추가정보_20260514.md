### 인하우스

**연회비 납부 적용 확인 시**
- 국문: `https://ihcf.or.kr/member/payment.php`, `https://ihcf.or.kr/member/application_payment_list.php`
- 영문: `https://ihcf.or.kr/eng/member/payment.php`, `https://ihcf.or.kr/eng/member/application_payment_list.php`
- 국문/영문 위 4군데 전부 반영되었는지 확인해야 됨

**테스트 진행 시 3가지 케이스 확인 필요**
- 해외만 3년 미만 자격증 취득된 경우
- 국내만 3년 미만 자격증 취득된 경우
- 해외·국내 모두 자격증 있으며 3년 미만인 경우
- 카드결제, 가상계좌 클릭 시 납부금액 반영 확인 필요

**평상시 적용되는 연회비**
- 한국 자격취득 연도 만 3년 이하인 경우: 10만원
- 해외 자격취득 연도 만 3년 이하인 경우: 10만원
- 한국·해외 자격취득 모두 있을 경우: 먼저 취득한 자격증 기준으로 적용
- 예: 한국 2023년 / 해외 2020년 → 연회비 15만원 (취득연도 만 3년 미만인 경우 5만원 할인 적용)
- 만 3년 기준 → 4년으로 적용됨 (25년도 기준으로 22~25년)

**매년 10~11월에만 적용되는 연회비** (자동 수정 불가 — 매년 수동 수정 필요)
- 한국 자격취득 연도 만 3년 이하인 경우: 15만원
- 해외 자격취득 연도 만 3년 이하인 경우: 15만원
- 한국·해외 모두 있을 경우: 먼저 취득한 자격증 기준 / 예: 한국 2023년 / 해외 2020년 → 연회비 20만원

**구글 플레이 스토어, 앱스토어 수정 시**: 구글플레이 콘솔, 앱스토어 개발 → 로그인하여 정보 수정 가능하며 고객사에 로그인 인증을 받아야 함
