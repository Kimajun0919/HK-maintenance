### 5-3. 인천감염병관리지원단 이북 작업 방법

- 유통물가와 작업 방식은 동일하나 추가 사항 참고
- 스킨: `111-S_A04_Kr` 사용
- 지난호보기 / 목차 페이지연결 기능 사용
  - 지난호보기: 기존 내용 참고하여 해당 주차로 생성될 이북 URL 넣으면 됨
  - 목차를 확인하여 페이지 연결
- 유통물가: NexBookAir / 본 컴퓨터 (인터넷 O)
- 인천감염병관리지원단: 이전 프로그램 그대로 작업 / 이북 컴퓨터 (인터넷 X)

**이북 컴퓨터 화면 전환** (인천감염병관리지원단 작업 시 필요)
- 화면 아래 가장 왼쪽 버튼 → 팝업 가장 아래 [기타] (톱니바퀴 두 개) → 입력소스를 HDMI로 변경

**지난호 보기 추가 예시**
```
ICDC WEEKLY_2024년 제21호(통권 제243호)
https://www.icdc.incheon.kr/pub/images/ebook/2421/index.html#page=1

ICDC WEEKLY_2024년 제22호(통권 제244호)
https://www.icdc.incheon.kr/pub/images/ebook/2422/index.html#page=1
```

**인천감염병관리지원단 작업 상세**
1. 작업할 파일 USB에 담기
2. 작업할 컴퓨터에 외장 및 마우스 USB 연결 후 프로그램 실행 (NexBook Builder CC)
3. 페이지(P) → 페이지 추가(A): 작업할 파일 불러와 누락 페이지 없는지 확인
4. 설정 변경:
   - 스킨 설정 → 스킨 관리 → `111-S_A04_Kr.jpg`
   - 환경 설정 → 출판 옵션 설정 → 오프라인에서 바로보기 / JPG(이미지형) 체크
   - 환경 설정 → 목차 관리 → 작업 문서에 맞는 목차와 페이지 등록 / 바탕화면 [목차] 메모장 확인
   - 환경 설정 → 지난호 관리 → 지난호 보기 사용 체크 후 우측 추가 버튼 클릭 → 제목과 URL 입력
5. 빠른제작 → 미리보기를 통해 빠진 것 없는지 검수
6. 빠른제작 → 출판하기 / 폴더명 주소와 동일하게 확인
   - 예: 연결 주소가 `https://www.icdc.incheon.kr/pub/images/ebook/2421/index.html#page=1`이었다면 폴더명을 `2421`로 하여 출판
7. 나스에 파일 업데이트: `\\hknas_2\hknas_2\VIP\04_유지보수(장기)\인천감염병관리지원단\ebook`
8. `/pub/images/ebook` 파일질라에 파일 업로드 (전체파일 폴더로 올려야 함, 접속 안될 시 가비아 로그인하여 방화벽 IP 추가)
9. 관리자페이지 → 게시판관리 / 이미 게시글은 고객사에서 올려놓은 상태 → ebook 링크 수정
10. 사용자페이지 내 보기 버튼 눌렀을 때 동작 확인 후 완료처리
